1. BoardMate의 핵심 페이지 중 하나인 "Meetups" 페이지 제작을 요청함
2. 사용자들이 모임을 탐색하거나 직접 생성할 수 있는 UI와 기능 중심의 설계를 목표로 함
3. 실 구현 단계에 앞서, Lovable에서 디자인 및 프로토타입 제작을 위한 구조를 명확히 전달함

---

## ChatGPT의 프롬프트 답변

### (영문)

Create the "Meetups" page for BoardMate.

🎯 **FUNCTION:**
Users can browse or create board game meetups.

🧭 **LAYOUT:**
- Top search bar (by location, keyword, game type)
- Map preview (Google Maps or Kakao Map placeholder)
- Grid of meetup cards:
  - Each card shows title, date/time, location, number of participants
  - "Join" and "Details" buttons
- Floating “+ Create Meetup” button at bottom right
- Meetup creation modal:
  - Title, date, time, location, max participants, description fields

🎨 **STYLE:**
- Rounded cards, clean layout
- Soft shadows and pastel color accents
- Compact responsive design (desktop + mobile)

---

### (한글 번역본)

BoardMate의 **“Meetups” 페이지**를 만들어줘.

🎯 **기능 요약:**
사용자가 보드게임 모임을 탐색하거나 새로 만들 수 있는 공간.

🧭 **레이아웃:**
- 상단 검색창 (위치, 키워드, 게임 종류별 검색)
- 지도 미리보기 (Google Maps 또는 Kakao Map 자리)
- 모임 카드 그리드:
  - 각 카드에는 제목, 날짜/시간, 장소, 참가 인원 표시
  - "참가하기" 및 "자세히 보기" 버튼 포함
- 우측 하단의 ‘+ 모임 만들기’ 플로팅 버튼
- 모임 생성 모달 창:
  - 제목, 날짜, 시간, 장소, 최대 인원, 설명 입력 필드 포함

🎨 **디자인 스타일:**
- 둥근 카드형 UI와 깔끔한 배치
- 부드러운 그림자와 파스텔 포인트 컬러
- 반응형 구조 (PC / 모바일 최적화)

---

### 💡 페이지 목적
BoardMate의 중심 기능 중 하나로, 실제 사용자가 보드게임 모임을 쉽게 탐색하고 직접 모임을 만들 수 있는 핵심 허브 역할을 함.  
Lovable 상에서 시각적으로 명확하고 직관적인 인터페이스로 제작하는 것을 목표로 함.
