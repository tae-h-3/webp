## 나의 웹 사이트 기획

### 1. 서비스 개요
**보드메이트**는 보드게임을 즐기는 사람들을 위해
- 주변 사람들과 같이 할 사람을 쉽게 찾고,
- 일정을 자동으로 조율하고,
- 보드게임 카페와 연동해 예약까지 완료할 수 있는 플랫폼

#### 핵심 목표: 
    보드메이트 : “보드게임 모임이 생기고, 유지되고, 즐겁게 돌아가도록 도와주는 웹사이트”

<hr>

### 2. 주요 사용자
1. 🧑‍🤝‍🧑 일반 사용자	
    - 보드게임 취미자, 친구/동호회 찾는 사람
2. 🎲 동아리/동호회 리더	
    - 정기 모임 일정, 출석, 홍보 수단 필요
3. 🏠 보드게임 카페	
    - 손님 유입, 예약 관리, 홍보 수단 필요

<hr>

### 3. 핵심 기능 설계
#### (1) 모임/매칭 기능
- 위치 기반 검색: 내 위치 기준 반경 5km 내 모임 표시 (카카오맵 API)
- 모임 생성:
    - 제목, 날짜, 게임 종류, 인원 수, 장소 설정
    - 참가자 모집 상태 실시간 표시
- 자동 매칭:
    - 사용자의 관심 게임, 위치, 가능 시간 기반 AI 추천
    - “나와 잘 맞는 플레이 성향” 분석 기능 (간단한 성향 테스트)
<hr>

#### (2) 일정표 & 출석 관리
- 모두의 일정표 통합:
    - 동아리별 캘린더
    - 참석 여부(✅참석 / ❌불참 / ❓미정) 투표
    - 카카오톡 알림 연동 (“3시간 뒤 모임 시작!”)
- 출석 관리:
    - 자동 출석 체크 (QR 코드 or GPS)
    - 누적 참여 횟수 통계화 (랭킹, 포인트)
<hr>

#### (3) 커뮤니티 기능
- 후기 / 사진 게시판
    - 모임 후기, 게임 리뷰, 전략 토론
- 좋아요 / 레벨 시스템
    - 참여도 기반 레벨 (ex. Lv.3 보드메이트 → “신입 가이드” 역할 가능)
- 추천 시스템
    - 자주 같이 노는 멤버 자동 추천
<hr>

#### (4) 보드게임 카페 연동
- 카페 등록/검색 기능
    - 위치, 영업시간, 보유 게임 리스트, 좌석 수 표시
- 예약 기능
    - 모임 생성 시 해당 카페로 자동 예약 연동
- 홍보 페이지 제공
    - 카페 사장이 자체 홍보글 작성 가능

<hr>

### 4. 예상 반응 & 확장성
- 보드게임 유저 기반이 폭발적으로 늘고 있음 (카카오 보드게임 카페 검색량 ↑)
- “같이 놀 사람 찾기 + 일정 관리”는 매우 보편적 문제
- 이후엔 TRPG / 카드게임 / 방탈출 모임 등으로 확장 가능
